EDB LDAP Sync is a collection of utilities and configurations for synchronizing Postgres with LDAP or Active Directory. Database administrators are able to schedule ldap2pg to synchronize Postgres users and groups with LDAP users and groups. EDB LDAP Sync replaces external scedulers, like cron, while using a similar syntax.
