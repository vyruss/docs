EDB Postgres Tuner is a PostgreSQL extension that automates 15+ years of EDB Postgres tuning experience. 
EDB Postgres Tuner enables you to apply tuning recommendations automatically, or view tuning recommendations and selectively apply them. It's now possible to successfully manage demanding Postgres databases without tuning expertise.
