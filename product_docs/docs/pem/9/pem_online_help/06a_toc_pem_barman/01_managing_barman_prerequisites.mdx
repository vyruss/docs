---
title: "Prerequisites for managing Barman"
---

Before adding a Barman server to the PEM console, install the following on the Barman host: 

- [Barman](https://docs.pgbarman.org/release/2.19/#installation)
- [Postgres backup API](https://www.enterprisedb.com/docs/supported-open-source/barman/pg-backup-api/)
